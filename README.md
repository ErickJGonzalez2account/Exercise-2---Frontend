## Tareas — Frontend

Aplicación de gestión de tareas construida con Nuxt 3 y TypeScript. (La paginacion se activa con 10 tareas)

## Tech Stack

- Nuxt 3 — framework Vue con SSR/SPA
- Pinia — manejo de estado
- SweetAlert2 — notificaciones toast
- TypeScript

## Requisitos

- Node.js 18+
- npm o pnpm

## Uso de IA

Consulté Claude como apoyo durante el proceso, más que nada para no trabarme en aspectos técnicos
